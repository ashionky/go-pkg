go-pkg
### 使用说明
```
克隆代码
git clone https://github.com/ashion89/go-pkg.git

本地配置代理
export GOPROXY="https://goproxy.cn,direct"

依赖下载
go mod tidy 
go mod vendor

修改配置
conf/dev.yml

打包:
go build 

启动:
./go-pkg

```

## api服务基础架构；修改配置后可直接启动
## pkg目录下为整理封装的各类工具方法且已经测试，可直接使用，相关目录下有对应的_test文件
